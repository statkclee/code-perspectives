### 재귀

재귀는 함수 자신을 반복해서 호출해서 점진적으로 동일한 문제를 작은 문제로 풀어나가는 기법이다.
모든 재귀 함수는 다음과 유사한 견본모양을 취한다:

~~~ {.python} 
def <함수명>:
    if <기본 조건>:   # 문제가 해결되었거나 거의 해결되었나?
        <기본 사례>        # 해결한다.
        return
   else:                   # 그렇지 않은 경우 문제가 해결되지 않았다면...
        <재귀 사례>   # ...문제를 조금 축소한다... 그리고...
        <재귀 호출>   # ...다시 문제를 풀려고 시도한다.
        return
~~~
